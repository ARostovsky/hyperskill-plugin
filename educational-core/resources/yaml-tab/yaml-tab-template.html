Course

course-info.yaml depicts the structure of a course:
Name Description Value
title Title of the course String
language Languages used for course materials Any of supported languages
summary Summary describing what the course is about String
programming_language Programming language of the course Any of supported programming languages
programming_language_version Optional version of programming language Version of programming language
environment Additional information about the course environment For example, Kotlin courses are supported for Android environment
content Folder names of lessons or sections included in the course List of folder names

Section

section-info.yaml depicts the structure of a section:
Name Description Value
custom_name Custom name for the section if it differs from its folder name String
content Folder names of the lessons included in the section List of folder names

Lesson

lesson-info.yaml manages the parameters of lessons:
Name Description Value
type Lesson type lesson
framework
custom_name Custom name for the lesson if it differs from its directory name String
content Directory names of tasks included in the section List of directory names

Task

task-info.yaml manages the parameters of a task:
Name Description Value
type Task type edu theory output ide code choice
custom_name Custom name for the task if it differs from its directory name string
files Files included in the task List of files included in the task
feedback_link Custom link for students to leave their feedback about the task E.g., Google Forms link
is_multiple_choice If choice task allows multiple correct answers true false
message_correct choice Custom message to be shown to a student in case of correct solution E.g., “Great job!” or “Fantastic!”
message_incorrect choice Custom message to be shown to a student in case of incorrect solution E.g., “Review how to define a function to solve this task”
options choice task answer variants List of options
